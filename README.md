## 🚀 Featured Projects

### 1. 🤖 Customer Churn Prediction — End-to-End SageMaker Deployment
> *Real ML in production, not just a notebook.*

Built a binary classification model (Random Forest / XGBoost) that achieved **92% accuracy** after feature engineering and hyperparameter tuning. The key wasn't just the model — it was deploying it:

- Packaged and deployed as a **real-time endpoint on Amazon SageMaker**
- Built a **Flask REST API** hosted on EC2 that consumes the SageMaker endpoint
- Used **S3** for data versioning and **CloudWatch** for live monitoring
- Entire pipeline is reproducible and documented on GitHub

**Stack:** `Python` `Scikit-learn` `SageMaker` `EC2` `S3` `Flask` `CloudWatch`

---

### 2. ⚡ Real-Time Sentiment Analysis — Serverless Pipeline
> *Zero servers. Real-time results. Production-grade architecture.*

Designed a fully serverless NLP pipeline from scratch — no EC2, no always-on servers, just events and functions:

- **AWS Lambda + API Gateway** processes incoming tweet streams in real time
- Pre-trained **BERT model (TensorFlow)** stored in S3 handles classification
- Results written to **DynamoDB**, trends visualised live in **QuickSight**
- Reduced inference latency by **~20%** through Lambda configuration tuning

**Stack:** `TensorFlow` `BERT` `Lambda` `API Gateway` `DynamoDB` `QuickSight` `S3`

---

### 3. 📊 Student Performance Analytics — Predictive Dashboard
> *Turning academic records into actionable institutional insights.*

An end-to-end analytics project covering the full data engineering + ML lifecycle:

- Data cleaning and EDA with **Pandas & NumPy** on messy academic records
- Regression model (Scikit-learn) achieving **15% accuracy improvement** over baseline
- **AWS Glue** ETL jobs + **Athena** for querying raw data stored in S3
- Interactive **QuickSight dashboard** delivered to stakeholders

**Stack:** `Pandas` `NumPy` `Scikit-learn` `AWS Glue` `Athena` `S3` `QuickSight`
